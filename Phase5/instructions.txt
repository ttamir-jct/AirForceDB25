הוראות כניסה והפעלה של מערכת ניהול התעופה:

### דרישות מערכת
1. התקן את Python מגרסה 3.8 ומעלה מ-https://www.python.org/ (ודא ש-pip כלול).
2. ודא שיש לך גישה למסד vנתונים PostgreSQL עם הגדרות מתאימות.

### התקנה
1. העתק את קובץ ה-gui-app.py לתיקייה במחשבך
2. פתח טרמינל או סביבת פיתוח (כגון IntelliJ IDEA) והיכנס לתיקייה בה ממוקם הקובץ.
3. הרץ את הפקודה הבאה כדי להתקין את מנהל ה-PostgreSQL:
pip install psycopg2-binary
4. פתח את קובץ gui-app.py בעורך טקסט או בסביבת פיתוח.
5. עדכן את פרטי החיבור למסד הנתונים ב-gui-app.py (host, database, user, password, port) עם הפרטים שלך ב-PostgreSQL.

### הפעלה
1. בתיקייה הנ"ל, הרץ את הפקודה הבאה בטרמינל:
python gui-app.py
או לחץ על כפתור "Run" אם אתה משתמש בIDE.
2. הממשק הגרפי ייפתח; השתמש בכפתורי הניווט כדי לגשת למסכים השונים (ניהול טייסים, מפעילים, מרכזים ולוח הבקרה למבצעים).

### הערות
- אם מופיעה שגיאה, בדוק את פרטי החיבור למסד הנתונים והגדרות הטבלאות.
